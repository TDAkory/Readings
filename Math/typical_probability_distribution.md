# 典型概率分布全解析

概率分布是描述随机变量取值规律的核心工具，广泛应用于统计分析、机器学习、工程建模等领域。本文按「离散型」和「连续型」分类，系统介绍最常用的典型概率分布，包括核心定义、概率公式、数字特征、适用场景及实际案例，兼顾理论严谨性与工程实用性。

## 一、离散型概率分布（随机变量取值为可数离散值）

离散型分布适用于「计数类场景」（如次数、个数、事件发生与否），核心特征是概率质量函数（PMF：\(P(X=k)\)）和累积分布函数（CDF：\(P(X≤k)\)）。

### 1. 伯努利分布（Bernoulli Distribution）

描述「单次二元试验」的结果（仅两种可能：成功/失败），是最简单的离散分布，也是二项分布的特例（\(n=1\)）

- **随机变量**：\(X\) 表示试验结果（1=成功，0=失败）；
- **参数**：\(p\)（成功概率，\(0≤p≤1\)），失败概率 \(q=1-p\)。
- **概率质量函数（PMF）**：  
  \(P(X=k) = \begin{cases}
  p & k=1 \\
  1-p & k=0
  \end{cases}\)
- **数字特征**：  
  期望 \(E[X] = p\)，方差 \(Var(X) = p(1-p)\)。

应用场景

- 单次抛硬币（正面=成功，\(p=0.5\)）；
- 单次抽奖（中奖=成功，\(p=中奖率\)）；
- 单次检测（阳性=成功，\(p=患病概率\)）。

### 2. 二项分布（Binomial Distribution）

#### 核心定义

描述「\(n\) 次独立伯努利试验」中成功次数的分布，核心是“独立重复、二元结果、固定概率”。

- **随机变量**：\(X\) 表示 \(n\) 次试验中的成功次数（\(X=0,1,...,n\)）；
- **参数**：\(n\)（试验次数），\(p\)（单次成功概率）。
- **概率质量函数（PMF）**：  
  \(P(X=k) = \binom{n}{k} p^k (1-p)^{n-k}\)，其中 \(\binom{n}{k} = \frac{n!}{k!(n-k)!}\)（组合数，表示从 \(n\) 次中选 \(k\) 次成功）；
- **数字特征**：  
  期望 \(E[X] = np\)，方差 \(Var(X) = np(1-p)\)。

应用场景

- 产品质检：100 件产品中不合格品数量（\(n=100\)，\(p=不合格率\)）；
- 市场营销：1000 名用户中点击广告的人数（\(n=1000\)，\(p=点击率\)）；
- 实验统计：50 次投篮中命中次数（\(n=50\)，\(p=命中率\)）。

特性

- 当 \(n\) 较大（\(n≥30\)）且 \(p\) 适中（\(0.1≤p≤0.9\)）时，可近似为正态分布 \(N(np, np(1-p))\)；
- 当 \(p\) 极小（\(p→0\)）且 \(n\) 极大（\(n→∞\)）时，可近似为泊松分布（\(\lambda=np\)）。

### 3. 泊松分布（Poisson Distribution）

描述「单位时间/空间内稀有事件发生次数」的分布，核心是“事件独立、发生率稳定、稀有”。

- **随机变量**：\(X\) 表示单位时间/空间内事件发生次数（\(X=0,1,2,...\)）；
- **参数**：\(\lambda\)（事件发生率，\(\lambda>0\)，即单位时间/空间内平均发生次数）。
- **概率质量函数（PMF）**：  
  \(P(X=k) = \frac{e^{-\lambda} \lambda^k}{k!}\)（\(e≈2.71828\) 为自然常数）；
- **数字特征**：  
  期望 \(E[X] = \lambda\)，方差 \(Var(X) = \lambda\)（期望=方差是泊松分布的核心识别特征）。

应用场景

- 稀有事件计数：单位时间内服务器收到的异常请求数（\(\lambda=平均异常请求率\)）；
- 事故统计：每月交通事故发生次数（\(\lambda=月均事故数\)）；
- 生物学：每毫升水中的细菌个数（\(\lambda=每毫升平均细菌数\)）。

案例

某医院急诊室每小时平均接收 3 个急救电话（\(\lambda=3\)），则“每小时接收 2 个电话”的概率为：  
\(P(X=2) = \frac{e^{-3} \cdot 3^2}{2!} ≈ \frac{0.0498 \cdot 9}{2} ≈ 0.224\)（22.4%）。

### 4. 几何分布（Geometric Distribution）

描述「一系列独立伯努利试验中，首次成功所需的试验次数」，分为两种常见定义（需注意场景区分）：

- 定义 1（试验次数）：\(X\) 表示首次成功前的试验次数（\(X=1,2,...\)）；
- 定义 2（失败次数）：\(X\) 表示首次成功前的失败次数（\(X=0,1,2,...\)）；
- **参数**：\(p\)（单次成功概率，\(0≤p≤1\)）。
- **概率质量函数（PMF）**：  
  \(P(X=k) = (1-p)^{k-1} p\)（前 \(k-1\) 次失败，第 \(k\) 次成功）；
- **数字特征**：  
  期望 \(E[X] = \frac{1}{p}\)，方差 \(Var(X) = \frac{1-p}{p^2}\)。

应用场景

- 抽样统计：连续抽样直到抽到不合格品的抽样次数（\(p=不合格率\)）；
- 游戏概率：连续抽奖直到中奖的次数（\(p=中奖率\)）；
- 设备测试：连续测试直到设备故障的测试次数（\(p=单次故障概率\)）。

案例

某抽奖活动中奖率 \(p=0.1\)，则“第 5 次抽奖才中奖”的概率为：  
\(P(X=5) = (0.9)^4 \cdot 0.1 ≈ 0.0656\)（6.56%）。

### 5. 负二项分布（Negative Binomial Distribution）

几何分布的推广：描述「一系列独立伯努利试验中，获得 \(r\) 次成功所需的试验次数」。

- **随机变量**：\(X\) 表示获得 \(r\) 次成功的试验次数（\(X=r, r+1, ...\)）；
- **参数**：\(r\)（目标成功次数），\(p\)（单次成功概率）。
- **概率质量函数（PMF）**：  
  \(P(X=k) = \binom{k-1}{r-1} p^r (1-p)^{k-r}\)（前 \(k-1\) 次中 \(r-1\) 次成功，第 \(k\) 次成功）；
- **数字特征**：  
  期望 \(E[X] = \frac{r}{p}\)，方差 \(Var(X) = \frac{r(1-p)}{p^2}\)。

应用场景

- 生产达标：连续生产直到产出 5 个合格产品的生产次数（\(r=5\)，\(p=合格率\)）；
- 体育竞技：球队直到赢得 3 场比赛的总场次（\(r=3\)，\(p=单场胜率\)）。

特性：当 \(r=1\) 时，负二项分布退化为几何分布（定义 1）。

## 二、连续型概率分布（随机变量取值为连续区间）

连续型分布适用于「测量类场景」（如长度、时间、浓度、误差），核心特征是概率密度函数（PDF：\(f(x)\)）和累积分布函数（CDF：\(P(X≤x) = \int_{-∞}^x f(t)dt\)），需注意：连续型变量取单个值的概率为 0（\(P(X=a)=0\)），仅关注区间概率。

### 1. 均匀分布（Uniform Distribution）

描述「随机变量在某一区间内均匀取值」的分布，所有子区间的概率与长度成正比，无偏好性。

- **随机变量**：\(X\) 表示区间 \([a,b]\) 内的连续值；
- **参数**：\(a\)（区间下限），\(b\)（区间上限），\(a < b\)。
- **概率密度函数（PDF）**：  
  \(f(x) = \begin{cases}
  \frac{1}{b-a} & a ≤ x ≤ b \\
  0 & 其他
  \end{cases}\)
- **累积分布函数（CDF）**：  
  \(F(x) = \begin{cases}
  0 & x < a \\
  \frac{x-a}{b-a} & a ≤ x ≤ b \\
  1 & x > b
  \end{cases}\)
- **数字特征**：  
  期望 \(E[X] = \frac{a+b}{2}\)（区间中点），方差 \(Var(X) = \frac{(b-a)^2}{12}\)。

应用场景

- 随机抽样：从 [0,1] 区间生成随机数（\(a=0, b=1\)，即标准均匀分布 \(U(0,1)\)）；
- 时间建模：公交车到站时间（假设在 [5,15] 分钟内均匀分布）；
- 测量误差：仪器测量的随机误差（在 [-ε, ε] 内均匀分布）。

案例

某公交车每 10 分钟一班，到站时间服从 \(U(0,10)\)，则“乘客等待时间≤3 分钟”的概率为：  
\(P(X≤3) = F(3) = \frac{3-0}{10-0} = 0.3\)（30%）。

### 2. 正态分布（Normal Distribution）

又称高斯分布（Gaussian Distribution），是自然界最常见的分布，描述「大量独立随机因素叠加后的结果」，核心特征是“钟形曲线、对称、集中在均值附近”。

- **随机变量**：\(X\) 可取任意实数（\(-∞ < X < +∞\)）；
- **参数**：\(\mu\)（均值，决定曲线中心位置），\(\sigma^2\)（方差，决定曲线陡峭程度），\(\sigma>0\)（标准差）。
- **概率密度函数（PDF）**：  
  \(f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}}\)；
- **标准正态分布**：当 \(\mu=0, \sigma=1\) 时，记为 \(X~N(0,1)\)，PDF 简化为 \(f(x) = \frac{1}{\sqrt{2\pi}} e^{-\frac{x^2}{2}}\)；
- **数字特征**：  
  期望 \(E[X] = \mu\)，方差 \(Var(X) = \sigma^2\)。

核心特性（68-95-99.7 法则），正态分布的取值概率高度集中在均值附近：

- 68% 的数据落在 \([\mu-\sigma, \mu+\sigma]\) 区间；
- 95% 的数据落在 \([\mu-2\sigma, \mu+2\sigma]\) 区间；
- 99.7% 的数据落在 \([\mu-3\sigma, \mu+3\sigma]\) 区间（3σ 原则，常用于异常值检测）。

应用场景

- 自然现象：人群身高、体重分布（受遗传、营养等多种因素叠加影响）；
- 工程测量：零件加工尺寸误差（受设备精度、环境温度等因素影响）；
- 机器学习：模型预测误差（大量独立误差源叠加）；
- 统计推断：中心极限定理（大量样本均值近似服从正态分布，是假设检验、置信区间的基础）。

### 3. 指数分布（Exponential Distribution）

描述「两次连续事件发生的时间间隔」的分布，核心特性是“无记忆性”（过去的时间不影响未来事件发生的概率），是泊松过程的时间间隔分布。

- **随机变量**：\(X\) 表示两次事件的时间间隔（\(X>0\)）；
- **参数**：\(\lambda\)（事件发生率，\(\lambda>0\)，即单位时间内平均发生次数，与泊松分布的 \(\lambda\) 一致）。
- **概率密度函数（PDF）**：  
  \(f(x) = \lambda e^{-\lambda x}\)（\(x≥0\)）；
- **累积分布函数（CDF）**：  
  \(F(x) = 1 - e^{-\lambda x}\)（表示“时间间隔≤x”的概率）；
- 数字特征：  
  期望 \(E[X] = \frac{1}{\lambda}\)（平均间隔时间），方差 \(Var(X) = \frac{1}{\lambda^2}\)。

核心特性：无记忆性，对于任意 \(s,t>0\)，有 \(P(X>s+t | X>s) = P(X>t)\)。  
例：某设备的故障间隔服从指数分布，若已正常工作 5 小时，那么再正常工作 3 小时的概率，与“从全新状态开始工作 3 小时”的概率相同（不受过去 5 小时的影响）。

应用场景

- 可靠性工程：电子设备的故障间隔时间（\(\lambda=故障率\)）；
- 排队论：顾客到达服务台的时间间隔（\(\lambda=平均到达率\)）；
- 服务系统：服务器处理请求的时间（\(\lambda=平均处理率\)）。

案例：某服务器处理一个请求的平均时间为 2 秒（\(\lambda=0.5\) 次/秒），则“处理时间≤3 秒”的概率为：  
\(P(X≤3) = 1 - e^{-0.5×3} ≈ 1 - 0.223 = 0.777\)（77.7%）。

### 4. 伽马分布（Gamma Distribution）

指数分布的推广：描述「\(k\) 个独立指数分布变量的和」（即“\(k\) 次事件发生的总时间间隔”），核心是“多事件间隔的累积分布”。

- **随机变量**：\(X\) 表示 \(k\) 次事件的总时间间隔（\(X>0\)）；
- **参数**：\(k\)（形状参数，\(k>0\)，表示事件次数，整数时为埃尔朗分布），\(\lambda\)（率参数，\(\lambda>0\)，事件发生率）。
- **概率密度函数（PDF）**：  
  \(f(x) = \frac{\lambda^k x^{k-1} e^{-\lambda x}}{\Gamma(k)}\)，其中 \(\Gamma(k)\) 为伽马函数（\(k\) 为整数时，\(\Gamma(k)=(k-1)!\)）；
- **数字特征**：  
  期望 \(E[X] = \frac{k}{\lambda}\)，方差 \(Var(X) = \frac{k}{\lambda^2}\)。

应用场景

- 排队论：顾客排队的总等待时间（\(k=顾客数\)，\(\lambda=服务率\)）；
- 可靠性工程：\(k\) 个串联设备的总故障时间（每个设备故障间隔服从指数分布）；
- 图像处理：图像灰度值分布（用于图像分割、降噪）。

特性

- 当 \(k=1\) 时，伽马分布退化为指数分布；
- 当 \(k\) 较大时，伽马分布近似为正态分布（中心极限定理）。

### 5. 贝塔分布（Beta Distribution）

描述「区间 [0,1] 内的连续概率分布」，核心优势是“灵活性强”（可通过参数调整为均匀、偏左、偏右、U 形等多种形状），常用于「概率的概率建模」（即“未知概率的先验分布”）。

- **随机变量**：\(X\) 表示区间 [0,1] 内的取值（常用于表示概率、比例）；
- **参数**：\(\alpha>0\)（成功相关参数），\(\beta>0\)（失败相关参数）。
- **概率密度函数（PDF）**：  
  \(f(x) = \frac{x^{\alpha-1} (1-x)^{\beta-1}}{B(\alpha,\beta)}\)，其中 \(B(\alpha,\beta) = \frac{\Gamma(\alpha)\Gamma(\beta)}{\Gamma(\alpha+\beta)}\) 为贝塔函数；
- **数字特征**：  
  期望 \(E[X] = \frac{\alpha}{\alpha+\beta}\)，方差 \(Var(X) = \frac{\alpha\beta}{(\alpha+\beta)^2(\alpha+\beta+1)}\)。

典型形状（参数影响）

- \(\alpha=1, \beta=1\)：退化为均匀分布 \(U(0,1)\)；
- \(\alpha>1, \beta>1\)：钟形分布（集中在均值附近）；
- \(\alpha<1, \beta<1\)：U 形分布（取值集中在 0 和 1 附近）；
- \(\alpha>1, \beta=1\)：右偏分布（取值集中在 1 附近）。

应用场景

- 贝叶斯统计：未知概率的先验分布（如中奖率 \(p\) 的先验分布，\(\alpha=过去中奖次数+1\)，\(\beta=过去失败次数+1\)）；
- 比例建模：产品合格率、市场占有率的分布（取值在 [0,1] 区间）；
- 机器学习： Dirichlet 分布的基础（用于主题模型、混合模型）。

## 三、离散型 vs 连续型分布核心区别

| 维度                | 离散型分布                          | 连续型分布                          |
|---------------------|-------------------------------------|-------------------------------------|
| 随机变量取值        | 可数离散值（如 0,1,2,...）          | 连续区间值（如 [a,b]、\(R\)）        |
| 核心函数            | 概率质量函数（PMF：\(P(X=k)\)）     | 概率密度函数（PDF：\(f(x)\)）        |
| 概率计算方式        | 求和（\(P(a≤X≤b) = \sum_{k=a}^b P(X=k)\)） | 积分（\(P(a≤X≤b) = \int_a^b f(x)dx\)） |
| 单个值概率          | \(P(X=a)≥0\)（可为正数）            | \(P(X=a)=0\)（仅关注区间概率）       |
| 典型应用            | 计数场景（次数、个数）              | 测量场景（时间、长度、浓度）         |

## 四、典型分布选型指南（快速匹配场景）

| 场景类型                | 推荐分布                          | 关键判断依据                          |
|-------------------------|-----------------------------------|---------------------------------------|
| 单次二元试验（成功/失败） | 伯努利分布                        | 仅两种结果，单次试验                  |
| \(n\) 次独立二元试验的成功次数 | 二项分布                        | 独立重复、二元结果、固定概率          |
| 单位时间/空间内稀有事件次数 | 泊松分布                        | 事件稀有、发生率稳定、独立            |
| 首次成功所需试验次数      | 几何分布                        | 关注“首次成功”，无记忆性              |
| \(r\) 次成功所需试验次数   | 负二项分布                      | 关注“多次成功”，几何分布的推广        |
| 区间内均匀取值            | 均匀分布                        | 无偏好性，概率与区间长度成正比        |
| 大量独立因素叠加的结果    | 正态分布                        | 钟形对称，适用于身高、误差、均值等    |
| 两次事件的时间间隔        | 指数分布                        | 无记忆性，适用于故障间隔、排队间隔    |
| \(k\) 次事件的总时间间隔   | 伽马分布                        | 指数分布的推广，适用于多事件累积间隔  |
| [0,1] 区间的概率/比例建模  | 贝塔分布                        | 灵活性强，适用于未知概率的先验分布    |

## 五、经典参考文献与工具

### 1. 理论参考

- 《概率论与数理统计》（茆诗松）：国内经典教材，详细覆盖所有典型分布的推导与应用；
- 《Introduction to Probability and Statistics for Engineers and Scientists》（Ross）：工程领域常用教材，侧重分布的实际建模；
- 《Statistical Distributions》（Johnson）：分布领域权威手册，含各类分布的特性与扩展。

### 2. 工程工具

- Python：`scipy.stats` 库（支持所有典型分布的 PDF/CDF 计算、随机采样，如 `scipy.stats.norm` 正态分布、`scipy.stats.poisson` 泊松分布）；
- R：内置 `distributions` 包（原生支持分布计算，如 `pnorm` 正态分布 CDF、`rbinom` 二项分布采样）；
- Excel：`NORM.DIST`（正态分布）、`POISSON.DIST`（泊松分布）等函数，适用于快速计算。
